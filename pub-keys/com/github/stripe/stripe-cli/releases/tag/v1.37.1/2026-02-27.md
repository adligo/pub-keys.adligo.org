# Stripe Uses sha256 checksums

- [Stripe Cli Linux Install Doc](https://docs.stripe.com/stripe-cli/install?install-method=linux)
- [v1.37.1 Downloads page](https://github.com/stripe/stripe-cli/releases/tag/v1.37.1)

```
mkdir stripe
cd stripe
wget https://github.com/stripe/stripe-cli/releases/download/v1.37.1/stripe_1.37.1_linux_x86_64.tar.gz
sha256sum stripe_1.37.1_linux_x86_64.tar.gz > stripe_1.37.1_linux_x86_64.tar.gz.sha256
diff stripe_1.37.1_linux_x86_64.tar.gz.sha256 ~/
diff stripe_1.37.1_linux_x86_64.tar.gz.sha256 ~/pub-keys.adligo.org/pub-keys/com/github/stripe/stripe-cli/releases/tag/v1.37.1/stripe_1.37.1_linux_x86_64.tar.gz.sha256
```

Also note this matches with the line from;

- [v1.37.1 sha256 checksums](https://github.com/stripe/stripe-cli/releases/download/v1.37.1/stripe-linux-checksums.txt)
