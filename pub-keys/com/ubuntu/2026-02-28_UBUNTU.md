# Install Ubuntu 2026-02-28

- [Download Ubuntu](https://ubuntu.com/download/desktop)

Verify the image

```
cd ~
git clone https://github.com/adligo/pub-keys.adligo.org.git
cd ~/Downloads
sha512sum ubuntu-24.04.4-desktop-amd64.iso > ubuntu-24.04.4-desktop-amd64.iso.sha512
cat ubuntu-24.04.4-desktop-amd64.iso.sha512
diff ubuntu-24.04.4-desktop-amd64.iso.sha512 ~/pub-keys.adligo.org/pub-keys/com/ubuntu/download/desktop/ubuntu-24.04.4-desktop-amd64.iso.sha512
```
